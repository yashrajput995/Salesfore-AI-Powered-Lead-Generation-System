
# Salesforce Developer Account Setup
## AI-Powered Lead Generation System

## 1. Overview
Before developing the Salesforce AI-Powered Lead Generation System, a Salesforce Developer Organization (Dev Org) must be created. A Dev Org provides a free environment where developers and administrators can configure Salesforce features, automate workflows, and test CRM applications.

---

## 2. Sign Up for Salesforce Developer Account

1. Navigate to the Salesforce Developer signup page:
   https://developer.salesforce.com/signup

2. Fill in the required details:

| Field | Value |
|------|------|
| First Name | Your First Name |
| Last Name | Your Last Name |
| Email | Your Email Address |
| Role | Developer |
| Company | College Name |
| Country | India |
| Postal Code | Your PIN Code |

### Username Format
The username must be unique and should follow this format:
Note: This does not need to be a real email address.
  yourname.college@organization.com  

3. Click **Sign Me Up** after completing the form.

---

## 3. Account Activation

1. Check the email inbox used during registration.
2. Open the Salesforce verification email.
3. Click **Verify Account**.

---

## 4. Set Password

After verification:

1. Create a secure password.
2. Select a security question.
3. Click **Change Password**.

You will now be redirected to the Salesforce Developer Org dashboard.

---

## 5. Developer Org Dashboard

Once logged in, you can access:

- Setup
- Object Manager
- Flow Builder
- Prompt Builder
- User Management
- Data Import Tools

This environment will be used to build the AI-Powered Lead Generation System.
